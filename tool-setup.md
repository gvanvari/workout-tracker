# Tool Setup Guide - macOS

Required tools for the Workout Tracker MVP.

## Installation

### Homebrew

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### Node.js & npm

```bash
brew install node
node --version    # Should be 18+
npm --version     # Should be 9+
```

### SQLite

```bash
brew install sqlite3
sqlite3 --version
```

### Git

```bash
brew install git
git --version
```
